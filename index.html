Secara sederhana, Algoritma adalah "resep" atau langkah-langkah logis untuk menyelesaikan masalah, sedangkan Pemrograman adalah proses menuliskan langkah-langkah tersebut agar dimengerti komputer.

  • Algoritma: Logika murni, tidak bergantung bahasa. (Contoh: Jika hujan, ambil payung).
  • Pemrograman: Implementasi teknis menggunakan kode (coding).

Analogi: Algoritma adalah partitur musik, Pemrograman adalah aktivitas memainkannya.

Contoh Algoritma (Logika):

  1. Mulai.
  2. Tentukan Panjang & Lebar.
  3. Hitung Luas = Panjang x Lebar.
  4. Tampilkan Luas.
  5. Selesai.
#Implementasi Algoritma
panjang = 10
lebar = 5
luas = panjang * lebar
print("Luas Persegi Panjang:", luas)

Variabel adalah wadah data. Di Python, tipe data ditentukan secara otomatis (Dynamic Typing).

Jenis Tipe Data Dasar:

  • int: Bilangan bulat (contoh: 10, -5).
  • float: Bilangan desimal (contoh: 3.14).
  • str: Teks/String (contoh: "Halo").
  • bool: Logika True/False.
nama = "Budi"       # String
umur = 25           # Integer
tinggi = 170.5      # Float
aktif = True        # Boolean

print(f"Halo {nama}, umurmu {umur}")

Fungsi utama untuk menampilkan output ke layar adalah print().

  • Standard: print("Teks").
  • f-string: Cara modern menyisipkan variabel ke teks dengan awalan huruf 'f'.
#Output Biasa
print("Hello World!")

#Output dengan f-string (Modern)
kota = "Jakarta"
suhu = 30
print(f"Suhu di {kota} adalah {suhu} derajat.")

Python dapat berfungsi sebagai kalkulator dengan operator matematika lengkap.

  • + - * / : Tambah, Kurang, Kali, Bagi.
  • // : Pembagian Bulat (tanpa koma).
  • % : Modulus (Sisa bagi).
  • ** : Pangkat.
a = 10
b = 3

print("Bagi:", a / b)   # Hasil: 3.33..
print("Bulat:", a // b) # Hasil: 3
print("Sisa:", a % b)   # Hasil: 1
print("Pangkat:", a ** 2) # Hasil: 100

Cara komputer mengambil keputusan berdasarkan kondisi tertentu.

  • if: Kondisi pertama.
  • elif: Kondisi tambahan jika yang pertama salah.
  • else: Pilihan terakhir jika semua salah.
nilai = 75

if nilai >= 90:
    print("Grade A")
elif nilai >= 70:
    print("Grade B") # Ini yang tereksekusi
else:
    print("Belajar lagi ya")
